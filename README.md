# Virtual Queue Management System

A QR-based virtual queue management system that allows customers to keep their place in line without physically waiting at the shop.

## Features

- QR-based queue joining
- Live queue position tracking
- Browser notifications
- Multi-chair support
- Admin-managed shops

## Tech Stack

Frontend:
- React
- Material UI

Backend:
- Node.js
- Express

Database:
- MongoDB Atlas

Realtime:
- Socket.IO
